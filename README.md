This project is a Blockchain-Based Data Sharing System built using Python and Streamlit.
It allows users to securely store structured data in blocks using hashing.


New!Click to edit

🚀 Features
User-based data input system
Multiple input fields:
Name
Data Type
Message
Blockchain structure with:
Index
Timestamp
Data
Hash
Previous Hash
Simple and interactive web UI
🛠️ Technologies Used
Python
Streamlit
Hashlib (SHA-256)
Datetime
📂 Project Structure
project/
│── app.py
│── model.py
│── utils.py

▶️ How to Run
1. Install dependencies
pip install streamlit
2. Run the application
streamlit run app.py
📥 Inputs in Application
👤 User Name
📂 Data Type (Text / Document / Image Info)
📝 Message
📦 Output
Displays blockchain with all blocks
Each block shows:
Index
Timestamp
Data
Hash
Previous Hash
